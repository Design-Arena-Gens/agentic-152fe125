# Playwright Word Add-in Roadmap

A curated 10-day learning plan that guides a beginner from zero to delivering a working Microsoft Word add-in automation suite with Playwright (Python) targeting WebView2 task panes. The site is built with Next.js and optimized for quick reference while studying.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the roadmap.

## Scripts

- `npm run dev` – Start the Next.js development server
- `npm run build` – Create a production build
- `npm run start` – Run the production build locally
- `npm run lint` – Lint the project with ESLint
- `npm run format` – Format files using Prettier

## Project Highlights

- Ten-day study plan with learning outcomes, practice tasks, and curated resources
- Additional sections covering daily routine, milestone checks, and tooling tips
- Responsive layout optimized for desktop and mobile reference

- Next.js 14 (App Router)
- React 18
